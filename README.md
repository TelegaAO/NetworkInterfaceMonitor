# Монитор сетевых интерфейсов

Проект помогает:
- Отслеживать статистики по сетевым интерфейсам
- Производит мониторинг входящего/исходящего трафика
- Обнаруживает неисправности сетевых интерфейсов

# Как запустить

1. Установите Python 3.6 или новее
2. Откройте командную строку и перейдите в папку с проектом
3. Установите зависимости:
pip install -r requirements.txt
4. Запустите программу:
python network_monitor.py
